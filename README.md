# Vatnafraedi_Jokulsa_a_fjollum
Lokaverkefni í Vatnafræði, Umhverfis og byggingarverkfræði, Háskóli Íslands

# Leiðbeiningar við keyrsu á verkefni:
Byrja á að hlaða niður gögnum frá LamaH-Ice gagnasett (Helgason og Nijssen, 2024), mikilvægt að nota nýjustu útgáfu, v1.5,
sjá hlekk: https://www.hydroshare.org/resource/705d69c0f77c48538d83cf383f8c63d6/
á vefsíðunni skal hlaða niður zip skránni lamah_ice.zip

Þar þarf að finna tvö gagnaskjöl sem heita báðar ID_45. Það er gert svona:
-Finna möppu með veðurgögnum: lamah_ice\A_basins_total_upstrm\2_timeseries\daily\meteorological_data\ID_45
  -Endurnefna ID_45, ID_45_veður
-Finna möppu með rennslisgögnum lamah_ice\lamah_ice\D_gauges\2_timeseries\daily\ID_45
  -Endurnefna ID_45, ID_45_rennsli

Hlaða niður kóða nefndur Python kóði og hlaða upp skránnum. Þá ætti að vera hægt að keyra innlestur gagna ásamt liðum 2-8
